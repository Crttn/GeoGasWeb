## 💡 Sobre el Proyecto

**GeoGasWeb** es una aplicación web interactiva de mapas diseñada con una estética minimalista y *glassmorphism*. Su objetivo es mostrar datos en tiempo real de estaciones de servicio clave (como **Repsol, BP y Disa**), ofreciendo una experiencia de usuario fluida y visualmente pulida. 

Este proyecto se integra mediante controladores alineados a DTO (Data Transfer Objects) para asegurar un manejo robusto de los datos del precio de la gasolina desde el backend.

## ✨ Características Principales

- **🗺️ Mapa Interactivo en Tiempo Real:** Navegación fluida por las estaciones de servicio disponibles.
- **⛽ Datos de Combustible y Precios:** Información actualizada de surtidores y capacidades.
- **🎨 UI/UX Premium:** Diseño minimalista con estilo *glassmorphism*, gráficos vectoriales planos y componentes visuales refinados.
- **🔖 Iconografía Personalizada:** Uso de iconos SVG únicos para marcas específicas de combustible.
- **⚡ Rendimiento Optimizado:** Carga ultrarrápida impulsada por **Vite** y tipado estricto con **TypeScript**.

## 🛠️ Tecnologías Utilizadas

* **Frontend:** TypeScript, Vite, HTML/CSS.
* **Backend / API:** Integración con backend propio (Spring Boot/Java para gestión de datos) a través de `server.ts`.
* **Diseño Visual:** Vectores planos, iconografía SVG personalizada y lineamientos de diseño minimalista.

## 📐 Filosofía de Diseño

El proyecto sigue una estricta filosofía de diseño **minimalista y de alta gama**. Se priorizan las interfaces limpias, el uso de transparencias (*glassmorphism*) y la usabilidad intuitiva, alineándose con estándares de diseño de productos digitales modernos.
